# Agent

## Agent Architecture Pattern

**Chosen:** Graph (LangGraph) — multi-step pipeline with conditional edges. Two compiled graphs: `profile_graph` for upload profiling (no LLM), `qa_graph` for Q&A (LLM code generation → sandboxed execution → prose formatting).

---

## LLM Provider & Model

| Node | Provider | Model ID | Rationale |
|------|----------|----------|-----------|
| plan_and_code | Gemini | gemini-2.0-flash | Cost-efficient code generation from schema+stats |
| format_response | Gemini | gemini-2.0-flash | Prose formatting of execution result |
| profile_data | none | — | Pure pandas computation, zero LLM calls |
| execute_code | none | — | Sandboxed exec(), zero LLM calls |

---

## Tools / Capabilities

| Tool | Input | Output |
|------|-------|--------|
| profile_csv | file path | profile dict (schema, stats, quality flags) |
| execute_pandas | code string + DataFrames dict | result string + optional Plotly JSON |

## AgentState TypedDict

```python
from typing import TypedDict, Any

class AgentState(TypedDict, total=False):
    session_id: str              # UUID — identifies the session in SQLite
    action: str                  # "profile" | "answer" | "error"
    uploaded_files: list[dict]   # [{file_id, filename, path, profile_json}]
    current_question: str | None # The user's current question
    generated_code: str | None   # Python code generated by plan_and_code
    execution_result: str | None # str(result) from exec() sandbox
    chart_json: dict | None      # Plotly figure as JSON-serialisable dict (or None)
    answer: str | None           # Formatted prose answer from format_response
    error: str | None            # Error message if any node fails
```

---

## Nodes

### profile_data(state: AgentState) → AgentState
- Reads the CSV at `state["uploaded_files"][0]["path"]` with pandas
- Computes per-column: dtype, null_count, null_pct, sample_values (first 3 non-null stringified)
- Numeric columns: min, max, mean, std, p25, p50, p75
- Categorical columns: top-5 value_counts
- Quality flags: >20% nulls → WARNING; all-null → ERROR; duplicate rows → WARNING
- Sets `action="profile"` in returned state
- NO LLM call

### plan_and_code(state: AgentState) → AgentState
- Loads conversation history from SQLite (last 10 turns for session_id)
- Builds prompt: system instruction + schema (col names + dtypes, NO raw rows) + stats + history + question
- Calls Gemini gemini-2.0-flash to generate pandas code (stores answer in `result`, optional chart in `fig`)
- Sets `generated_code` in returned state
- On Gemini error: sets `error`

### execute_code(state: AgentState) → AgentState
- Loads all uploaded CSVs as DataFrames (keyed by filename stem)
- Runs `generated_code` via exec() in restricted namespace: `{dfs, pd, np, go, px}`
- Captures `result` → `execution_result` (str), `fig` → `chart_json` (Plotly JSON dict)
- On exception: sets `error` with traceback summary
- Timeout: 30 seconds

### format_response(state: AgentState) → AgentState
- Calls Gemini gemini-2.0-flash with `execution_result` + `current_question`
- Returns plain prose answer for a non-technical reader
- Sets `answer`

### handle_error(state: AgentState) → AgentState
- Sets `action="error"`
- Sets `answer = f"I encountered an error: {state['error']}. Please try rephrasing your question."`

## Graph Edges

```
profile_graph:
  [START] → profile_data → [END]

qa_graph:
  [START] → plan_and_code ─── error? ──→ handle_error → [END]
                           └── ok ──→ execute_code ─── error? ──→ handle_error → [END]
                                                    └── ok ──→ format_response → [END]
```

Conditional edge functions:
- `after_plan(state)`: returns "handle_error" if state.get("error"), else "execute_code"
- `after_execute(state)`: returns "handle_error" if state.get("error"), else "format_response"

## Runner Entry Points

```python
def run_profile(session_id: str, file_id: str, filename: str, file_path: str) -> dict:
    """Profile a freshly uploaded file. Returns profile dict."""
    initial: AgentState = {
        "session_id": session_id,
        "action": "profile",
        "uploaded_files": [{"file_id": file_id, "filename": filename, "path": file_path}],
    }
    final = profile_graph.invoke(initial)
    return final["uploaded_files"][0]["profile_json"]

def run_question(session_id: str, question: str, uploaded_files: list[dict]) -> dict:
    """Answer a natural-language question. Returns {answer, chart_json}."""
    initial: AgentState = {
        "session_id": session_id,
        "action": "answer",
        "current_question": question,
        "uploaded_files": uploaded_files,
    }
    final = qa_graph.invoke(initial)
    return {"answer": final.get("answer"), "chart_json": final.get("chart_json")}
```

---

## Memory Strategy

- Within a session: conversation history in SQLite `messages` table
- Each Q&A turn loads last 10 messages from DB for LLM prompt context
- No cross-session memory; all data deleted on session end
- AgentState is ephemeral per invocation — no LangGraph checkpointing in Phase 1

---

## Error Handling

| Failure | Response |
|---------|----------|
| Gemini API error | route to handle_error → return error message to user |
| CSV read error | route to handle_error |
| exec() exception | catch, set error, route to handle_error |
| exec() timeout (> 30s) | kill, set error, route to handle_error |
| Missing session | raise 404 HTTP error at API layer before invoking graph |

---

## Observability

- structlog structured logging on every node entry/exit: session_id, node name, duration_ms
- Log Gemini token usage (prompt_tokens, completion_tokens) per call to stdout
- LangSmith tracing: optional — enabled when LANGCHAIN_TRACING_V2=true + LANGCHAIN_API_KEY set in .env

## Graph Assembly (pseudocode)

```python
from langgraph.graph import StateGraph, END
from src.graph.state import AgentState
from src.graph.nodes import profile_data, plan_and_code, execute_code, format_response, handle_error
from src.graph.edges import after_plan, after_execute

# Profile graph (upload path)
pb = StateGraph(AgentState)
pb.add_node("profile_data", profile_data)
pb.set_entry_point("profile_data")
pb.add_edge("profile_data", END)
profile_graph = pb.compile()

# QA graph (question path)
qb = StateGraph(AgentState)
qb.add_node("plan_and_code", plan_and_code)
qb.add_node("execute_code", execute_code)
qb.add_node("format_response", format_response)
qb.add_node("handle_error", handle_error)
qb.set_entry_point("plan_and_code")
qb.add_conditional_edges("plan_and_code", after_plan, {"execute_code": "execute_code", "handle_error": "handle_error"})
qb.add_conditional_edges("execute_code", after_execute, {"format_response": "format_response", "handle_error": "handle_error"})
qb.add_edge("format_response", END)
qb.add_edge("handle_error", END)
qa_graph = qb.compile()
```
